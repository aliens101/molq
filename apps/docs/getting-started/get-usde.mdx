---
title: "Get USDe on Mantle"
description: "Routes available for obtaining or bridging USDe before using MolQ."
---

MolQ accepts the canonical USDe token on Mantle:

```text
0x5d3a1Ff2b6BAb83b63cd9AD0787074081a52ef34
```

You can open the **Get USDe** dialog in the application to access supported
external routes:

<CardGroup cols={3}>
	<Card title="Stargate" icon="bridge" href="https://stargate.finance/">
		Bridge supported assets to Mantle.
	</Card>
	<Card title="Relay" icon="arrows-rotate" href="https://relay.link/">
		Use a crosschain transfer route to Mantle.
	</Card>
	<Card title="Merchant Moe" icon="arrow-right-arrow-left" href="https://merchantmoe.com/">
		Swap supported Mantle assets for USDe.
	</Card>
</CardGroup>

<Warning>
	These are independent third-party applications. Verify the destination chain, token address,
	route, fees, price impact, and received amount before signing. MolQ does not control their
	contracts or interfaces.
</Warning>

You also need MNT in the same wallet to pay Mantle transaction fees.
